# Developer Portfolio

GitHub Pages에 바로 배포할 수 있는 반응형 개발자 포트폴리오입니다.

## 로컬에서 보기

`index.html`을 브라우저로 열거나 간단한 정적 서버를 실행하세요.

```powershell
python -m http.server 8000
```

## GitHub Pages 배포

1. 이 폴더를 GitHub 저장소의 기본 브랜치에 push합니다.
2. 저장소의 **Settings → Pages**로 이동합니다.
3. **Deploy from a branch**를 선택하고 기본 브랜치의 `/ (root)`를 지정합니다.

## 수정 포인트

- `index.html`: 이름, 소개, 프로젝트, 링크 및 이메일
- `styles.css`: 색상, 간격, 반응형 레이아웃
- `script.js`: 모바일 메뉴와 스크롤 등장 효과
